# Write Away by Dr.J

An AI-powered dissertation and academic writing review platform created by Dr. Joel M. Stancer.

## Vision

Write Away by Dr.J helps doctoral and graduate students receive developmental, mentor-style writing feedback while progressing toward dissertation completion.

## Planned Features

- DOCX and PDF uploads
- AI-generated dissertation feedback
- Developmental coaching in Dr. J's voice
- APA and scholarly writing support
- Chapter-specific analysis
- Revision comparison tools
- Dissertation persistence support

## Stack

- Next.js
- React
- TailwindCSS
- OpenAI API
- Mammoth
- PDF-Parse
